# Dbase frontend

## 사용 기술

- REACT(vite)
- styled-components
- react-router-dom
- react-markdown(예정)

## 버전

- node.js **(22.16.0)**  
  [node.js 설치(LTS)](https://nodejs.org/ko/download)  
  `npm -v` -> `version number`
- pnpm **(10.12.1)**  
   `npm install -g pnpm`  
   `pnpm -v` -> `version number`

## 실행

**pnpm**

```
    git clone https://github.com/jiyoonyee/DBase-frontend.git
    cd DBase-frontend
    pnpm i
    pnpm run dev
```
